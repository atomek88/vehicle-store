# Junkyard Tracker

Vehicle inventory management system with layered architecture, type-safe validation, and comprehensive test coverage.

**Tech Stack**: Next.js 16 • React 19 • TypeScript (strict) • Tailwind CSS v4 • Zod • Vitest • Playwright

> See [SUMMARY.md](./SUMMARY.md) for detailed architecture and design documentation.

---

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000/vehicles](http://localhost:3000/vehicles)

### Build

```bash
npm run build       # Production build
npm run start       # Run production build
```

---

## Testing

### Unit & Integration Tests (Vitest)

```bash
npm run test                    # Run all tests
npm run test -- --watch         # Watch mode
npm run test:coverage           # Coverage report
```

### E2E Tests (Playwright)

```bash
npm run test:e2e                          # Headless mode
npm run test:e2e -- --ui                  # Playwright UI
npm run test:e2e -- --headed              # Browser visible
npm run test:e2e tests/e2e/filters.spec.ts  # Specific test
```

### Linting

```bash
npm run lint
```
